Prologue

From Bad Method to Bad Structure

In Chapter 1, we showed that the Transformer's layer-by-layer update is formally equivalent to Forward Euler integration of a learned nonlinear dynamical system:

Chapter 1 Result: Forward Euler Update
$$x_{k+1} = x_k + f_k(x_k)$$

The critique there was about the method: Forward Euler is a crude, unstable integrator that amplifies errors, drifts from the true trajectory, and overcommits to early signals.

But Chapter 2 is about something deeper. Even if you replaced Euler with a perfect integrator, the system being integrated is itself mathematically unconstrained in ways no real physical memory system ever is.

This chapter asks a precise question: what must be mathematically true of any system that evolves memory stably? Then we check, systematically, whether Transformers satisfy any of those conditions. The answer is: they satisfy none, not by accident, but by construction.

§ 1
Section I

Transformer as a General Nonlinear Dynamical System

The Transformer's residual stream defines a state variable \( x_k \in \mathbb{R}^d \) at each layer \( k \). The full update rule is a state transition map:

General State Transition
$$x_{k+1} = T_k(x_k), \quad \text{where} \quad T_k(x) = x + f_k(x)$$

This is a well-defined nonlinear discrete-time dynamical system. It obviously is one. The question is whether it is a well-behaved one.

What do we want from a memory system? The final hidden state \( x_L \) should faithfully represent relevant semantic content, without being corrupted by noise, without amplifying early ambiguities, without spiraling into runaway interpretations. Mathematically, this requires:

The Five Requirements for Stable Memory

1. Not amplify perturbations uncontrollably, contraction or non-expansion

2. Forget noise faster than signal, selective dissipation

3. Prevent runaway modes, spectral stability

4. Evolve within bounded energy, Lyapunov stability

5. Admit a well-defined energy function that decreases, Lyapunov function existence

We examine each condition in turn, first rigorously, then applied to the Transformer update.

§ 2
Section II

Contraction Mappings and Semantic Stability

The Mathematical Definition

A mapping \( T : \mathbb{R}^d \to \mathbb{R}^d \) is a contraction if there exists \( \alpha \in [0,1) \) such that for all \( x, y \):

Contraction Condition
$$\|T(x) - T(y)\| \leq \alpha\,\|x - y\|, \quad \alpha < 1$$

The Banach Fixed Point Theorem guarantees that any contraction has a unique fixed point, and iterating from any initial condition converges there. This is the mathematical skeleton of stable memory.

Intuition: Two Interpretations of a Sentence

Imagine two hidden states \( x \) and \( y \): one correct interpretation of an ambiguous sentence, one slightly wrong. In a contractive system, \(\|T(x)-T(y)\| < \|x-y\|\): the two interpretations get closer with each layer. The system forgets the initial ambiguity. In a non-contractive system, the two interpretations diverge, and the model commits increasingly to whichever path received the first perturbation.

Why the Transformer Is Not a Contraction

For the Transformer map \( T(x) = x + f(x) \), compute the difference between two outputs:

Transformer Output Difference
$$T(x) - T(y) = \underbrace{(x - y)}_{\text{original gap}} + \underbrace{(f(x) - f(y))}_{\text{nonlinear addition}}$$

Even small nonlinear differences get added, not damped. Nothing enforces \( \|T(x)-T(y)\| < \|x-y\| \). The Jacobian of the attention mechanism can have eigenvalues arbitrarily larger than 1. Training may accidentally produce approximate contraction in some regimes, but the architecture imposes no such guarantee.

Contractive System

Two nearby states \( x, y \) evolve toward each other. Noise is forgotten. Ambiguity resolves. The system has a unique attractor. Memory is stable.

Transformer (Residual Map)

\( T(x) = x + f(x) \), the residual adds nonlinear differences rather than damping them. Small semantic ambiguity can grow across layers. No structural damping exists.

§ 3
Section III

Lyapunov Stability and the Absence of Bounded Energy

The Framework

A function \( V : \mathbb{R}^d \to \mathbb{R}_{\geq 0} \) is a Lyapunov function for \( x_{k+1} = T(x_k) \) if:

Lyapunov Conditions
$$V(x) > 0 \;\forall x \neq 0, \quad V(0) = 0, \quad V(T(x_k)) \leq V(x_k) \;\forall k$$

If such a function exists, the system is Lyapunov stable: the state's "energy" cannot increase. If \( V(T(x_k)) < V(x_k) \) strictly, states converge to the fixed point, asymptotic stability.

Intuition: Marble in a Bowl

A marble placed anywhere in a bowl will roll toward the bottom. The bowl's shape defines a Lyapunov function, potential energy. No matter where you place the marble, energy decreases monotonically. The bowl enforces convergence through its geometry. Physical memory systems have this bowl-like structure in state space.

Why the Transformer Has No Lyapunov Function

Take \( V(x) = \|x\|^2 \) as the simplest candidate. Compute under the Transformer update:

Norm Growth Under Transformer Update
$$\|x_{k+1}\|^2 = \|x_k + f_k(x_k)\|^2 = \|x_k\|^2 + 2\langle x_k,\, f_k(x_k)\rangle + \underbrace{\|f_k(x_k)\|^2}_{\geq\, 0\text{ always}}$$

The last term is always non-negative. For norm to decrease we would need \( 2\langle x_k, f_k(x_k)\rangle \) to be sufficiently negative, meaning the correction must point strongly opposite to the current state. The attention mechanism is designed to add relevant information, which almost always makes this inner product positive. The \( \|f_k\|^2 \) term acts like perpetual energy injection at every layer.

"No structural mechanism ensures energy decay. Even if training data prefers stability, the equation does not enforce it."

Layer Normalization: A Partial, Surface-Level Patch

Layer Norm rescales \( x_k \) to unit norm within a sub-component. But it does not enforce that the semantic content evolves contractively. It clips the magnitude but not the direction. Two hidden states pointing in increasingly divergent directions are not made closer by normalization; they are merely scaled. The Lyapunov violation is about directional divergence in high-dimensional state space, not scalar magnitude.

§ 4
Section IV

Spectral Instability and Runaway Modes

Local Linearization

Near any hidden state \( x_k \), linearize the Transformer update via its Jacobian \( J_{f_k} = \frac{\partial f_k}{\partial x}\big|_{x_k} \):

Local Linearization → Eigenvalues of Transition Matrix
$$x_{k+1} \approx (I + J_{f_k})\,x_k \implies \mu_i = 1 + \lambda_i(J_{f_k})$$

For discrete stability all eigenvalues must lie strictly inside the unit circle: \( |\mu_i| < 1 \). This requires every eigenvalue of \( J_{f_k} \) to lie inside a disk of radius 1 centered at \( -1 \). Nothing in the architecture, loss function, or training enforces this extraordinarily tight constraint.

Spectral Instability → Observable Failures

Runaway interpretation: the model amplifies its first strong semantic commitment across layers, unable to revise it.

Narrative commitment: in long-context generation, early tokens get amplified so strongly that later tokens cannot update the global interpretation.

Hallucination persistence: once a hallucinated fact enters the residual stream with a large eigenvector component, it propagates and grows, overriding contradictory evidence.

§ 5
Section V

What Real Physical Systems Do Differently

The canonical model for memory in physical and engineered systems is the linear state-space model:

Continuous Linear State-Space System
$$\frac{dx}{dt} = Ax + Bu, \quad y = Cx + Du$$

The matrix \( A \in \mathbb{R}^{n \times n} \) is the system matrix, and its spectrum is everything. For stability, all eigenvalues of \( A \) must have strictly negative real parts, \( \text{Re}(\lambda_i(A)) < 0 \). This is not an emergent property of training. In engineered systems, \( A \) is designed to have this spectral property. In biological circuits, evolutionary pressure achieves it.

Physical systems are also dissipative: they continuously lose energy unless actively driven:

Dissipativity Condition
$$\frac{d}{dt}V(x) \leq -\alpha V(x) + \beta\|u\|^2, \quad \alpha > 0$$

Without input, energy decreases exponentially. With input, energy is bounded by input power. Memory is retained only when actively sustained, and this is exactly right. The Transformer makes no such distinction between "sustaining existing memory" and "adding new information." It just adds.

Physical / LDS Memory

Memory decays unless driven. Spectrum is engineered for selective retention. Energy bounded by input power. Dissipativity ensures stability. Forgetting is structural.

Transformer Memory

Memory accumulates without bound. No spectral engineering. Energy grows with depth. No dissipation mechanism. Forgetting is accidental, not guaranteed by structure.

§ 6
Section VI

State Space Models: The Structured Alternative

The structural deficiencies of the Transformer motivated Structured State Space Models (S4) and successors (S5, Mamba), which explicitly build stability into the architecture. S4 begins with the continuous-time LDS and discretizes via the matrix exponential, not Euler:

Transformer (Forward Euler)

$$x_{k+1} = (I + J_{f_k})\,x_k$$

Eigenvalues are \(1+\lambda_i\). No stability guarantee. Can expand freely.

S4 (Matrix Exponential)

$$x_{k+1} = e^{A\Delta t}\,x_k + \bar{B}\,u_k$$

Since \(\text{Re}(\lambda_i(A)) < 0\), all eigenvalues of \(e^{A\Delta t}\) satisfy \(|\cdot| < 1\). Stability is structural.

The S4 update is a genuine contraction. It is Lyapunov-stable. It has spectral structure that selectively retains memory in low-decay modes and forgets noise in high-decay modes. These are not training artifacts; they are structural guarantees baked into the matrix \( A \).

"The Transformer accumulates. The SSM flows. One adds whatever it learned directly to memory. The other evolves memory according to stable internal dynamics."
§ 7
Synthesis

Five Conditions. Zero Satisfied.

Every single condition for stable memory evolution is either absent or only surface-patched by the Transformer. S4, designed with dynamical systems theory in mind, satisfies all of them structurally:

Condition Math Requirement Transformer S4 / SSM
Contraction \(\|T(x)-T(y)\|\leq\alpha\|x-y\|,\;\alpha<1\)< /td> ✗ Not enforced ✓ Structural
Lyapunov Stability \(\exists V: V(T(x))\leq V(x)\) ✗ Norm grows ✓ By design
Spectral Stability \(|\mu_i|<1\) for all eigenvalues ✗ No constraint ✓ By construction
Dissipativity \(\frac{dV}{dt}\leq-\alpha V+\beta\|u\|^2\) ✗ No decay term ✓ Via stable A
Bounded Evolution \(\|x_k\|\leq C\) for all \(k\) ✗ LayerNorm only ✓ Exponential bound

Transformers work extraordinarily well on tasks we give them, partly because training data is structured, finite depth limits how much instability accumulates, and softmax provides partial damping. But these are empirical patches, not structural guarantees. The moment you push outside training distribution, longer contexts, adversarial prompts, genuine long-horizon memory, the structural deficiencies surface as observable failures.

Hallucination, context drop, narrative commitment, and token bias are all fingerprints of the same underlying problem: a state evolution system with no contraction, no Lyapunov function, and no spectral constraint.
